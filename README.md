<h1>Tic Tac Toe (The Odin Project)</h1>

<p>
  A browser-based <strong>Tic Tac Toe</strong> game built as part of
  <strong>The Odin Project’s JavaScript Course</strong>. The focus of this project is clean JavaScript
  structure, using modules/factories, and separating game logic from DOM/UI code.
</p>

<h2>Features</h2>
<ul>
  <li>Play <strong>Tic Tac Toe</strong> in the browser with a clickable 3×3 grid</li>
  <li>Two-player local gameplay (Player X vs Player O)</li>
  <li>Prevents overwriting occupied cells</li>
  <li>
    Detects game outcomes:
    <ul>
      <li>X wins</li>
      <li>O wins</li>
      <li>Tie</li>
    </ul>
  </li>
  <li>Displays game status in the UI</li>
  <li>Stops interaction once the game ends (board disabled)</li>
</ul>

<h2>Built With</h2>
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript (modules / IIFE pattern, factory functions)</li>
</ul>


<h2>Project Requirements (The Odin Project)</h2>
<p>This project is based on The Odin Project’s Tic Tac Toe assignment:</p>
<ul>
  <li>Use JavaScript to control the game flow</li>
  <li>Store gameboard in an array</li>
  <li>Build game logic first, then connect it to the DOM</li>
</ul>
